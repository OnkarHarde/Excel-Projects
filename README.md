# 📊 Ferns and Petals Sales Analysis (Excel Project)

## 📌 Project Overview

This project presents an end-to-end **Sales Analysis Dashboard** built using Microsoft Excel. The goal of this project is to analyze sales performance, customer behavior, and product trends to derive meaningful business insights.

The dashboard provides a clear and interactive visualization of revenue across different dimensions such as **categories, occasions, time, and geography**.

---

## 🎯 Objectives

* Analyze overall revenue performance
* Identify top-performing products and categories
* Understand customer spending behavior
* Discover sales trends across months and days
* Evaluate performance based on different occasions

---

## 📂 Dataset Description

The dataset consists of multiple tables:

* **Customers** – Customer details
* **Orders** – Order transactions including date, quantity, and revenue
* **Products** – Product information and categories

---

## ⚙️ Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers (for dynamic filtering)
* Data Cleaning & Transformation
* KPI Cards

---

## 📊 Dashboard Features

### 🔹 Key Metrics

* **Total Revenue:** ₹35,20,984
* **Average Customer Spend:** ₹2,937

### 🔹 Visual Insights

* **Revenue by Category** – Identifies highest selling product categories
* **Revenue by Month** – Tracks monthly performance trends
* **Revenue by Occasion** – Highlights sales during events like Diwali, Holi, Valentine’s Day
* **Revenue by Country/City** – Shows geographical distribution of orders
* **Top 5 Products** – Best performing products based on revenue
* **Revenue by Day** – Weekly sales pattern analysis

---

## 📈 Key Insights

* Certain categories contribute significantly higher revenue than others
* Sales peak during specific occasions (e.g., festivals and special days)
* Weekly trends indicate higher sales on specific days
* A few top products generate a large portion of total revenue (Pareto effect)
